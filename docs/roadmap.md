# Roadmap

## Phase 1 - Foundation

- Create project structure.
- Add API gateway scaffold.
- Add web portal scaffold.
- Add local infrastructure services.
- Document the initial product direction.

## Phase 2 - Patient Registry

- Persist platform patient index records.
- Add PostgreSQL migrations.
- Add external identifier mapping.
- Prepare OpenMRS synchronization adapter.
- Add audit logs for patient operations.

## Phase 3 - Laboratory Workflow

- Add exam request model.
- Add specimen model.
- Add laboratory result model.
- Prepare OpenELIS integration adapter.
- Generate DiagnosticReport-compatible responses.

## Phase 4 - Research Layer

- Add study registry.
- Add consent validation.
- Add pseudonymization workflow.
- Prepare LabKey export adapter.
- Add dataset versioning metadata.

## Phase 5 - Governance and Analytics

- Add Keycloak integration.
- Add role-based access control.
- Add MinIO document storage.
- Add dashboards and metrics.
- Add LGPD-oriented audit reports.
